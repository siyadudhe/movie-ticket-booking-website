# A Responsive Booking Website
 MovieMania is a sleek and fully responsive website designed to provide users with information on their favorite movies. The website dynamically adapts to various screen sizes, ensuring a seamless browsing experience on desktops, tablets, and mobile devices.
 <p>Technologies Used:

HTML: For structuring the website content.

CSS: For styling and layout, including Flexbox and Grid for responsive design.

JavaScript: For interactivity, dynamic content, and API integration.</p>
